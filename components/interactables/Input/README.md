# Description
Generic input component.

# Usage
```vue
  <InteractablesInput
    v-model="something"
    size="small"
    input-kind="number"
    type="dark"
  />
```

# Props

**text** Input display text

**size** Input display size, honors bulma.io sizes

**input-kind** Abstracts generic input type attribute

**label-text** Optional label appended to input
